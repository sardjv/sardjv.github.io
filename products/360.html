SARD 360 is a simple, easy to use multi-source feedback module designed to work hand in hand with other SARD products. Based on the GMC patient, colleague and self-assessment questionnaires, it is built with the necessary flexibility to reflect the differing requirements of individual roles within NHS trusts.

Features

  • Full self-assessment, patient and colleague feedback system
  • Advanced real-time reporting functionality
  • Results checked against national benchmarking data
  • Accurately reflects differing 360 requirements
  • Built around standard GMC questionnaires
  • Instantly email forms to users and non-users
  • Rapid data entry of paper forms
  • Complete audit trail
  • Advanced administration and user management system
  • Secure 24/7/365 web access from anywhere

Advantages

  • Simple, intuitive design promotes user engagement
  • Improve compliance and engagement rates via comprehensive reporting features
  • System access from multiple devices
  • Progress tracking against set requirements
  • Ability to reflect difficult roles within trusts
  • Familiarity through custom branding
  • Designed with a major NHS trust alongside software specialists
  • Best-in-class doctor and technical support
  • Quick, simple to use and user-driven, administration-light approach
  • Access all 360 information via a single interface
